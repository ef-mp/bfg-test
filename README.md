## Доступные команды

Для установки и запуска необходим `Node.js` не ниже версии `8.10` и не выше `12.16.2`. <br>
`Npm` не ниже версии `5.6` <br>
Для установки выполнить команду `npm install`.

### `npm start`

Запускает dеv-сервер на http://localhost:3000

### `npm run build`

Запускает сборку
